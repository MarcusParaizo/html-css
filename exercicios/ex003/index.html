Itálico e Negrito

Para se usar o negrito basta utilizar o comando b(não semântico) ou strong(semântico)

Para se usar itálico podemos usar o i(não semântico) ou o em(semântico)

Outras formatações

Texto marcado

Ex:Para marcar o texto, basta usar o comando mark

Texto deletado

Para usar texto deletado, basta usar o comando del

Texto inserido

Para usar o texto inserido, basta usar o comando ins

Texto sobrescrito

Ex: X2 + 3

Texto subscrito

Ex: H2O

Citação simples

Basicamente, é o aspas, porém de forma semântica, basta usar o q

Citação completa

Para destacar o texto, e uma citação, se usa o blockquote
Exemplo:

Batatiha quando nasce, espalharrama pelo chão...

Abreviação

Para mostrar o significado de uma abreviação, passano o mouse por cima dela, basta usar o abbr
Ex: HTMLe CSS
